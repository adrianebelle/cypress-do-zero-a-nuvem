#Curso Cypress

Este é um projeto para aprender Cypress e conhecer os comandos e funções básicas.

Pre-requisito
Instalação do Node.js, cypress para executar o projeto

Usei as versões mais atuais.

Instalação

npm install cypress --save-dev

##Testes
É possível executar os testes no desktop ou mobile.

###Desktop
Para executar os testes npm run cy:open - executa os testes em headless

###Mobile
npm run cy:open:mobile - abre a aplicação Cypress em um mobile
